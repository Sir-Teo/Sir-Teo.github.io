"Picobot"

Recently, I have been working to solve this fun game Picobot.

The goal of this game is to have the "picobot" run through every empty spaces on the map "empty room", as presented below.

At first glance, it seems difficult because the robot can never when to go north/east/south/west, plus, what happens if the robot encounters a "road branch off". What's more, there are as much as 16 possible surroundings.

Then i googled the solution of these kind of maze and discovered a very subtle solution---the "right hand rule". The right hand rule states that one can always get out of a maze if left hand always holds the wall

Here is a sample solution to this maze:

						0 *X** -> E 3
						0 *E** -> X 1
						1 X*** -> N 0
						1 N*** -> X 2
						2 **X* -> W 1
						2 **W* -> X 3
						3 ***X -> S 2
						3 ***S -> X 0
